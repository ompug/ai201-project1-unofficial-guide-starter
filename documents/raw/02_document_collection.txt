The project needs at least ten specific documents, pages, threads, or files. The README should name the domain and identify the sources clearly enough that another person could locate them. Local text files count when the file names and descriptions are specific.

The source list should show variety. For this AI201 Project 1 survival guide, the documents cover domain selection, planning, ingestion, chunking, retrieval, grounded generation, query interface, evaluation, stretch features, and AI usage transparency. Together they answer different questions instead of repeating the same checklist ten times.

Students should skim every document before writing pipeline code. Skimming reveals whether the corpus consists of short posts, long guides, checklists, or mixed formats. That decision directly affects the chunk size and overlap.
